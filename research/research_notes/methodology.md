# Research Methodology

## Objective
Conduct comprehensive evidence-based research on Tom Rohrböck through his Facebook profile at https://www.facebook.com/tom.rohrbock.1

## Research Approach

### 1. Profile Analysis
- **Basic Information Collection**: Name, location, education, work history
- **Profile Visibility Assessment**: Public vs private information
- **Timeline Analysis**: Post history, activity patterns
- **Visual Documentation**: Screenshots of key profile sections

### 2. Network Analysis
- **Friends Network**: Identify key connections and relationships
- **Family Connections**: Document family members and relationships
- **Social Groups**: Track group memberships and affiliations
- **Interaction Patterns**: Analyze comment and activity patterns

### 3. Content Analysis
- **Posts Activity**: Content themes, posting frequency, engagement
- **Photos & Media**: Visual content analysis, tagged photos
- **Interests & Pages**: Liked pages, interests, affiliations
- **Communication Style**: Language patterns, sentiment analysis

### 4. Evidence Collection
- **Screenshot Documentation**: Visual evidence of all findings
- **Data Organization**: Structured filing system for all evidence
- **Source Verification**: Cross-reference information where possible
- **Timestamp Documentation**: Record when information was collected

## Ethical Considerations
- Only collecting publicly available information
- Respecting Facebook's terms of service
- Not attempting to access private content
- Maintaining research integrity and accuracy

## Tools & Methods
- Browser MCP tools for navigation and documentation
- Systematic screenshot capture
- Structured note-taking
- File organization for evidence management

## Data Management
- All evidence stored in organized folder structure
- Clear file naming conventions
- Research log maintained throughout process
- Regular backup of collected data
